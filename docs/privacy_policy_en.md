# Scorebook Privacy Policy

Last updated: July 19, 2026

Scorebook is an offline score tracking app. It does not create accounts and uses no advertising, analytics or user tracking. Profile, friend, photo, game and score data is stored only on the user's device and is not transmitted to the developer.

Photo library access occurs only when the user chooses a profile or friend photo. Backup and sharing operations are initiated by the user and transfer data to a destination selected through the operating system share sheet.

Users can permanently remove all local data from Settings > Data and privacy > Delete all data. Uninstalling the app also removes its local data from the device.

For privacy questions, use the developer support contact shown on the store listing.
